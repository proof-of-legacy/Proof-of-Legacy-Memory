The algorithm

One formula. Pure physics.

No boosts. No artificial multipliers. No favoritism by RAM type. The only variable that matters is your memory's raw latency.

score = 1 / latency_ns
Slower RAM → higher latency → more work per step → valid block
Faster RAM → lower latency → less work per step → still valid block
Every generation has its natural place. No exceptions.
01
Build the Memory DAG
Each epoch seeds a pseudorandom buffer from chain state — starting at 256 MB, doubling every epoch. Every access is a real DRAM read. CPU cache is completely bypassed. No shortcut possible.
02
Sequential memory walk
100,000 steps per nonce, each depending on the previous. Impossible to parallelize. The only bottleneck is physical DRAM latency. Up to 4 threads allowed per miner.
03
Physics enforces the score
Your score is 1 divided by your average latency in nanoseconds. DDR2 at ~3800 ns scores differently than DDR5 at ~500 ns — but both contribute honestly to the network.
Any generation. Any machine.

Your RAM mines. Period.

PoLM doesn't pick winners by RAM type. Every generation competes on its own physics. Older RAM is slower — it does more work per step. Newer RAM is faster — it processes more nonces. The network needs both.

Why PoLM

Not another SHA-256 clone. Consensus grounded in the physics of DRAM.

⚗️
Physics-based security
DRAM latency is a physical property — it cannot be faked, virtualized, or replicated in ASIC. The variation between RAM generations is ~8×. In SHA-256, ASICs are 100,000× faster than CPUs. PoLM is naturally ASIC-resistant by design.

Built-in hardware cycle
Every ~138 days the DAG doubles. By Epoch 7, mining requires 1 TB of RAM — creating demand for a new hardware category: dedicated RAM mining arrays, exactly like GPUs transformed Bitcoin mining. A new industry emerges.

Start in 3 commands

Any PC. Any RAM. DDR2 to DDR5. No special hardware, no pool, no fee. Your machine mines directly against the network.

Linux / macOS
# 1. Clone the repository
git clone https://github.com/proof-of-legacy/Proof-of-Legacy-Memory.git
cd Proof-of-Legacy-Memory && bash scripts/install.sh

# 2. Generate your wallet (saves 24-word seed phrase)
python3 polm_bip39.py new "My Wallet"

# 3. Start mining — any RAM type, no arguments needed
python3 polm.py miner https://polm.com.br/api/ YOUR_ADDRESS
