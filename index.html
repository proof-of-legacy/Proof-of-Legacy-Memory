The Algorithm

One formula. Pure physics.

No boosts. No multipliers. No favoritism by RAM type. The only variable that matters is your memory's raw latency.

Score formula
score = 1 / latency_ns
Slower RAM → higher latency → higher score/step → valid block
Faster RAM → lower latency → more nonces per second → valid block

Every generation has its natural place. No exceptions. No losers.
01
Build the Memory DAG
Each epoch seeds a pseudorandom 256 MB+ buffer from chain state. Every access is a real DRAM read — CPU cache completely bypassed. No shortcut possible.
02
Sequential memory walk
100,000 steps per nonce, each depending on the previous. Impossible to parallelize. The only bottleneck is physical DRAM latency. Max 4 threads per miner.
03
Physics enforces the score
Score = 1 / latency_ns. The network measures real hardware latency. DDR2 at ~3800ns and DDR5 at ~500ns both contribute honestly. It cannot be faked.
